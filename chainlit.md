# LegalRAG

**Интеллектуальная система анализа российского законодательства**

---

### Архитектура

| Модуль | Технология |
|--------|-----------|
| Поиск | Hybrid BM25 + Semantic + Neo4j Graph |
| Генерация | Gemini 3 Flash + LangGraph CRAG |
| Верификация | 5-уровневая проверка фактов |
| Хранение | PostgreSQL + ChromaDB + Redis |

### Как работает

1. **Анализ запроса** — определение типа вопроса и стратегии поиска
2. **Гибридный поиск** — одновременно по ключевым словам, семантике и графу знаний
3. **Оценка релевантности** — автоматический отбор лучших фрагментов
4. **Генерация ответа** — с цитатами из конкретных статей
5. **Верификация** — проверка корректности цитат и юридической точности
