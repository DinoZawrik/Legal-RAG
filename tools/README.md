# 🛠️ Tools - Инструменты для диагностики и анализа

После оптимизации папка `tools` содержит **объединенные утилиты** для эффективной работы с системой.

## 📦 Структура (Оптимизированная)

### 🔍 **PDF Analysis Suite** (`pdf_analysis_suite.py`)
**Объединенный инструмент для анализа PDF файлов**
- ✅ Тестирование различных методов извлечения (PyPDF2, PyMuPDF, pdfplumber)
- ✅ Поиск критических фраз в документах
- ✅ Сравнение результатов извлечения с базой данных
- ✅ Генерация детальных отчетов по качеству извлечения

```bash
# Полный анализ PDF файлов
python -m tools.pdf_analysis_suite --mode=full

# Только аудит извлечения
python -m tools.pdf_analysis_suite --mode=audit

# Тест методов извлечения
python -m tools.pdf_analysis_suite --mode=extraction
```

### 🔍 **Search Diagnostics Suite** (`search_diagnostics_suite.py`)
**Объединенный инструмент для диагностики поиска и анализа контента**
- ✅ Базовая диагностика поиска по ключевым словам
- ✅ Продвинутый анализ покрытия документов
- ✅ Быстрый аудит качества контента
- ✅ Анализ проблем сегментации
- ✅ Генерация рекомендаций по улучшению

```bash
# Полная диагностика поиска
python -m tools.search_diagnostics_suite --mode=full

# Базовая диагностика
python -m tools.search_diagnostics_suite --mode=basic

# Продвинутая диагностика
python -m tools.search_diagnostics_suite --mode=advanced

# Аудит контента
python -m tools.search_diagnostics_suite --mode=content
```

### 📊 **System Monitoring** (`system_monitoring.py`)
**Объединенные утилиты для мониторинга системы и базы данных**
- ✅ Проверка статуса ChromaDB через HTTP API
- ✅ Мониторинг файловой системы и важных файлов
- ✅ Анализ логов и обнаружение ошибок
- ✅ Проверка подключения к PostgreSQL
- ✅ Анализ производительности базы данных
- ✅ Статистика обработки документов

```bash
# Полный мониторинг системы
python -m tools.system_monitoring --mode=full

# Только статус системы
python -m tools.system_monitoring --mode=status

# Только база данных
python -m tools.system_monitoring --mode=database
```
